    # 1.关于网页的修改

    首先要先说一些基础的点：

    # 1.各种常见的标签的叫法及相应的应用

    div是最常见的标签，你平时浏览网页百分之90都是由div组成的，你可以把它想象成一个盒子，盒子里面自然要装东西，由此就演发出以下的标签，a标签、video标签、img标签、p标签各种h标签等等ps：你如果以后还有修改网页的作业，比较常见的就是修改这些了

    a标签是用来做链接跳转的，一版你会改到的就是herf属性和target属性，herf是代表要跳转的链接，target代表跳转的方式，比如当target是_blank 即target="_blank"时，跳转的是在新标签页打开，如果不设置则是在当前标签打开。

    p标签一般是来存放文字的（如果正规来写就应该这样，奈何很多人开发的时候不规范，所以div存放文字也有可能会）

    video标签是用来存放视频的，一版是设置它的src属性，代表视频的地址（还有一种可能是放在video标签下的source 标签里，不过也是设置src），这个标签的属性你要记住两个，一个是muted 代表静音播放，另一个是autoplay 代表自动播放，直接写上就好了，比如：

     <video autoplay muted  src="./wj"></video>
    1

    对了，在设置文件地址的时候，./代表这个文件在html文件的同一个文件夹内，../代表在html的上一个文件夹里面

    img标签记得src代表图片地址就好了

    h标签是代表标题，后面带的数字越小代表标题越大

    # 2.上次总结的修改网页的方法

    分成以下几步 1.在对应的网页打开控制台

    2.第二步，点击左上角的图标，选择要修改的元素，在控制台找到它

    3.第三步，找到这个标签，然后找里面的关键字，比如在这里我可以搜索“更多两个字”

    4.第四步，在你的软件里搜索这个关键字，找到代码里这个标签，然后进行修改

    这样就完成了网页的修改啦

    # 2023.10.5 这部分是属于预习课

    我来讲讲递归，要怎么才能看出来函数是一个递归函数呢？

    递归有一个非常重要的特点那就是自己调用自己

    首先要先讲讲函数是怎么调用的 先写一个很简单的函数

    function add(a,b){
    return a+b
}

    先讲一些关于js的基础知识，首先明白两个点就好了，一个是变量，一个是函数。变量就是值的意思，顾名思义变量就是会变的量值，一般需要进行声明，也就是对变量的定义。

    在题目里你应该能经常看到，let a=某某，var a = 某某 count a=某某

    这就是在进行变量的声明，一般声明的类型有字符串（也就是文字的意思），也就是带有""的,比方说let a = "wj"，就是声明了一个变量a，a的值为wj，还有一种是数字，比方说let a = 202，也就是声明了一个变量a，a的值为202

    要识别js里一个函数，首先就看它定义，function相当于声明了一个函数，add是函数的名字，再看括号内容，（a,b），相当于这个函数需要传入两个参数，第一个参数名为a，第二个参数名为b，然后再看{}里面的，很简单的return了a+b。看到有return表明这个函数是有返回值的，而我举得这个例子表明这个函数返回了a+b,然后要怎么调用呢，调用的时候就直接在声明函数之后写 函数名（传的参数）就可以了，比方说

    add(1,2)
    1

    像这样就调用成功了，然而这个函数是有返回值的，所以一般需要用一个值接收比如：

    var a = add(1,2)
    1

    聪明的你是不是知道了，现在这个a就是等于1+2也就是3

    而函数里也可能不return值，而是进行一些操作，总之，函数里可以做任何事情。

    接着我们看infiniteAaa，这个函数就有传入值了，它把传入的值命名为input，ps：函数传入值的命名是可以随意命名的。接着它对这个值进行了一些操作，首先input +="a"的意思是在这个值后面一段文字，文字的量是a，因为在上一个函数传入的值是一个空文字，所以空的文字+a这段文字是不是就会变成a了，ps：如果传的值是数字，而且+的值也是数字，就会变成加减法了。然后它又return了它本身，infiniteAaa，而且它把进行拼接后的值又传了进去，所以现在就会变成infiniteAaa('a'),那么再进行操作的话就会变成aa，而且你可以看到它在自己调用它自己，一开始也说了递归有一个非常重要的特点那就是自己调用自己,所以它是一个递归函数，所以题目问你的这一段是不是一个递归函数，那就是了。

    接着题目问你是怎么看出来的，你就可以说调用自身的函数就是递归函数，infiniteAaa调用了它自身，所以它是一个递归函数

    还有一个额外的考点，不知道你们题目会不会出的很细，就是你可以看到像这段函数，它仅仅只有调用，而且是一直调用，我们都知道计算机的计算速度是很快的，像这样一直的调用而没有停止，可能一秒钟计算机就会调用很多次了。而且呢，doSomething它return了infiniteAaa这个递归函数，但是肯定会报错的，因为根本return不出来，我返回一个一直在调用自身的函数怎么可能会返回的出来，就像我一直在绕一个圈，没有出口，怎么走的出来呢？所以递归函数一定要有判断终止的条件。比如说：

    在for后面的括号一般要带3个参数，用;隔开，第一个参数是这个循环的起始计数值，像下面的例子中我们定义它的起始是从0次开始，第二个参数是满足循环的条件，我们在下面的例子中写到它循环的条件是起始的计数值<5，这是不是很像一个if的条件呢，第三个参数是每次循环结束后起始计数值的变化情况，我们在下面的例子中让每次循环后起始计数值+1。当循环开始时，首先会判断第二个参数是否成立，成立则进行{}里面的操作，比如第一次循环时起始计数值是0，那肯定是满足小于5的，而满足时就会进行操作，当操作结束后也就是一次循环结束时，起始计数值就会进行for（括号里第三个参数的操作），也就是i会变成i+1,0就会变成1了。那么就进行第二次循环了，而1还是小于5的，这时候是不是还会进行循环哦，所以类推，进行了n次循环后当i不再小于5时，就结束循环啦，这时候a会等于多少呢？
